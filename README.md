# backend-challenge
Challenge backend in javascript

## Technologies used
- NodeJS (asynchronous event-driven JavaScript runtime)
- Express (REST Server)

## Commands
### Run the project
1. Clone the repository
```bash
git clone https://github.com/meeighteen/backend-challenge
```

2.  Install the dependecies
```bash
npm install
```

3. Run the project
```bash
 npm run dev
```
